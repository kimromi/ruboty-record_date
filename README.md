# Ruboty::RecordDate

Record date handler for Ruboty.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruboty-record_date'
```

## Usage

```
# record date(monitoring all comments)
record-date <body>

# date record list
@ruboty record list <date>

# date record delete
@ruboty record list <date> <time>
```
